CSS
===

Pegasus lets you choose from a number of CSS frameworks and themes, including two that use
`Bootstrap 5 <https://getbootstrap.com/>`_, one that uses `Tailwind CSS <https://tailwindcss.com/>`_,
and one that uses `Bulma <https://bulma.io/>`_.
Every Pegasus theme works on desktop and mobile can can be easily customized to match your specific site/brand.

Read on for help on choosing a CSS theme, and documentation on working with CSS in Pegasus.
